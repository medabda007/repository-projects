Système de Gestion des Courriers (KITAB)


MANUEL D’UTILISATEUR





1. Introduction
1.1 Présentation de l'application

KITAB est une solution de Gestion Électronique de Courrier et l’archivage de documents, offre un ensemble des fonctionnalités specifiques y compris l'enregistrement, le suivi et le traitement électronique des courriers entrants et sortants

1.2 Objectif du manuel

Ce guide vous aidera à tirer le meilleur parti de notre application KITAB en vous guidant à travers les différentes fonctionnalités.

2. Connexion à l'Application

Pour vous connecter à l'application, vous devez disposer d'un compte utilisateur et d'un mot de passe. Si vous ne disposez pas de compte, vous pouvez en créer un en contactant l'administrateur de l'application.

Pour vous connecter, saisissez votre nom d'utilisateur et votre mot de passe dans les champs prévus à cet effet. Cliquez ensuite sur le bouton "Connexion".

3. Interface Utilisateur
3.1. Accueil

Lorsque vous vous connectez à l'application, vous arrivez sur l'écran d'accueil. Dans le corps de cette page d’accueil, nous apercevons l’ensemble des courriers en cours, et en tête de la page, nous apercevons le menu principal, le moteur de recherche et un formulaire de filtrage des courriers en fonctions des ou statut.

3.2. Menu principal

En tête de chaque page se trouve un menu principal qui représente l’ensemble des liens permettant d’ajouter un courrier entrant ajouter un courrier sortant, lister les courriers entrants ou sortants, changer la langue, déconnecter la session d’utilisateur en cours.

3.3. Moteur de recherche

Sous menu principal il y a un moteur de recherche qui permet de chercher toutes les courriers existes par (objet, référence, numero d’enregistrement ou l’expediteur).

On peut aussi filtrer les courrier en fonction des dates ou statut.

4. Réception du courrier

Pour recevoir du courrier, vous devez d'abord configurer un scanner. Une fois le scanner configuré, vous pouvez scanner le courrier entrant.

4.1. Ajouter un courrier entrant

Pour ajouter un courrier entrant vous cliquez sur le bouton ‘Ajouter un courrier entrant’ en haut de la page dans le menu principal.

Si vous cliquez sur le boutoun 'Ajouter un courrier entrant' un formulaire de saisie de courrier s'affiche.

A partir de ce formulaire vous pouvez également saisir manuellement les informations relatives au courrier, et validez l’enregistrement en cliquant sur le bouton ‘Enregistrer’ en bas du formulaire. Une fois courrier enregistrer l'application affiche alors les informations relatives au courrier dans la page suivante:

4.2. Modifier un courrier entrant

Pour modifier les information de courrier deja enregistrer, vous cliquez sur le bouton "Modifier".

Une fois vous cliquez sur le bouton 'modifier', l'application va afficher le formulaire de modification suivant:

A partir de ce formulaire vous pouvez modifier les informations, et validez l’enregistrement en cliquant sur le bouton ‘Enregistrer’ en bas du formulaire.

4.3. Supprimer un courrier

On peut aussi supprimer un courrier enrant au cas de besoin, en cliquant sur le bouton "supprimer".

NB: On peut pas modifier ou supprimer le courrier entant que n'est pas encore soumi, c'est a-dire desque on clique sur le boutoun "soumettre" on peut plus modifer ou suprrimer le courrier.

4.4. Soumettre un courrier

Dans la liste des courriers sélectionnez le courrier que vous souhaitez soumettre, en cliquant sur le bouton voir pour afficher les informations de courrier dans la page suivante:

En bas de la page vous cliquez sur le boutton 'soumettre'. une fois cliquer sur cette bouton les deux boutons 'modifier' et 'supprimer' seront desactivé on peut plus les utiliser.

5. Traitement du courrier

Une fois le courrier reçu, vous devez le traiter. Cela peut inclure l'attribution de tâches aux destinataires, la prise de notes et l'enregistrement des actions effectuées.

Pour traiter un courrier, double-cliquez sur celui-ci dans la liste des courriers. L'application affiche alors les informations relatives au courrier.

Vous pouvez alors effectuer les actions suivantes :

  • Retourner le courrier : cliquez sur le bouton "Retourner le courrier" pour renvoyer le courrier au secretariat pour le courriger.
  • Annoter à : cliquez sur le bouton "Annoter à" pour attribuer le couurier et donner le main a quelqu'un de le traiter.
  • Traitement du courrier : cliquez sur le bouton "Traitement du courrier" pour enregistrer le tratiement du courrier et le cloturer.

6. Autres fonctionnalités

Les applications KITAB proposent généralement d'autres fonctionnalités, telles que :

  • La recherche de courrier : vous permet de rechercher des courriers en fonction de critères tels que l'expéditeur, le destinataire, la date ou le sujet.
  • Le téléchargement des courriers : vous pouvez telecharger des courriers dans un format PDF.

7. Conclusion

Ce guide vous a donné un aperçu de l'utilisation de l'application "KITAB". Pour plus d'informations, vous pouvez contacter l'administrateur de l'application.
